This is an H1

类 Atx 形式

用1-6个“#”标识一至六级标题

markdown标题标准写法写法需要在”#”和后面字符之间加一个空格，如果不加空格，有些引擎就解析不了。

列表篇

列表项目标记通常是放在最左边，但是其实也可以缩进，最多 3 个空格，项目标记后面则一定要接着至少一个空格或制表符。

有序列表

number.数字不能省略但可无序，点号之后的空格不能少

1. 有序列表
2. 有序列表
3. 有序列表
8. 有序列表

输出结果：

  1. 有序列表
  2. 有序列表
  3. 有序列表
  4. 有序列表

    无序列表

    无序列表使用星号、加号或是减号作为列表标记
  • 符号之后的空格不能少
  • -，+，*效果一样，但不能混合使用，因混合是嵌套列表

    嵌套列表

    -+*可循环使用，但符号之后的空格不能少，符号之前的空格也不能少
    - 嵌套列表
 + 嵌套列表
 + 嵌套列表
  - 嵌套列表
   * 嵌套列表
- 嵌套列表

输出结果：

  • 嵌套列表
    • 嵌套列表
    • 嵌套列表
    • 嵌套列表
      • 嵌套列表
  • 嵌套列表

表格篇

这里有一个空行
| 表头 | 表头 | 表头 |
|:-----|:---:|----------:|/*居左，居中，居右*/
|文本内容| **`是`** |文本内容|
|文本内容| **`是`** |文本内容|
|文本内容| **`是`** |文本内容|
  1. 最上面一行是表格第一列的值。第二行的冒号位置决定表格内文本的对齐方式。有水平居中、水平靠左对齐、水平靠右对齐三种。这两行是必须有的。
  2. 切记表格要与上面的文本内容空一行。否则解析不出来。
  3. 每列的宽度是根据对应列里最长的文本来决定的。
  4. 文本与“|”之间可以有n(n>=0)个空格，多余的会被自动忽略。

输出结果：

表头 表头 表头
文本内容 是 文本内容
文本内容 是 文本内容
文本内容 是 文本内容

特殊字符篇

转义：在特殊字符前加上反斜杠”\”。

段落与换行篇

段落：

段落的前后要有一个以上的空行（空行的定义是显示上看起来像是空的，便会被视为空行。比方说，若某一行只包含空格和制表符，则该行也会被视为空行）。普通段落不该用空格或制表符来缩进。

换行：

  1. Markdown中不能直接通过回车来换行，需要使用\
    。
  2. 在换行处先按入两个以上的空格然后回车。

发表于 2017-04-23

节点

表示网络中的一个连接点，每个节点都是一个对象

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="utf-8">
    <title>Shopping list</title>
</head>
<body>
    <h1>What to buy</h1>
    <p title="a gentle remainder">Don't forget to buy this stuff.</p>
    <ul id="purchases">
        <li>A tin of beans</li>
        <li class="sale">Cheese</li>
        <li class="sale important today">Milk</li>
    </ul>
</body>
</html>

一、元素节点（element node）–DOM的原子

  1. 标签的名字就是元素的名字
  2. <html>元素是唯一没有被包含在其他元素里的元素，是节点树的根元素

    二、文本节点（text node）–包含文本的节点

    在XHTML文档里，文本节点总是被包含在元素节点的内部

    三、属性节点（attribute node）–用于对元素做出更具体的描述

    属性节点总是被包含在元素节点中
    <p title="a gentle remainder">Don't forget to buy this stuff.</p>中
    <p>是元素节点，Don't forget to buy this stuff.是文本节点，title="a gentle remainder"是属性节点

获取元素

一、getElementById

返回一个有着与给定id属性值的元素节点对应的对象

  • 语法：document.getElementById("id")//单引号或双引号都可以

  • 例子：document.getElementById("purchases")

    二、getElementsByTagName

    返回一个对象数组

  • 语法：element.getElementsByTagName("tag")

  • 例子：document.getElementsByTagName("li")
  • getElementsByTagName允许把一个通配符（*）当作参数
  • alert(document.getElementsByTagName("*").length;//弹出文档中元素节点的总数

getElementById与gteElementsByTagName结合使用：

var shopping = document.getElementById("purchases");
var items = shopping.getElementsByTagName("li");
for (var i = 0;i < items.length;i++) {
    alert(typeof items[i]);
}

三、getElementsByClassName

返回一个具有相同类名的元素的数组

由于这个方法还比较新，只有较新的浏览器才支持它，某些DOM实现中可能还没有，使用时要小心

  • 语法：getElementsByClassName(class)
  • 例子：document.getElementsByClassName("sale")

alert(document.getElementsByClassName("important sale").length); //1

  1. 可以查找带有多个类名的元素，只需指定多个类名，用空格分隔；
  2. 类名的实际顺序不重要
  3. 元素还可以带有更多的类名

兼容新老浏览器的函数：

//不支持多个类名
function getElementsByClassName(node,className) {
//node表示DOM中树的搜索起点
    if (node.getElementsByClassName) {
        //使用现有方法
        return node.getElementsByClassName(className);
    }
    else {
        var results = new Array();
        var elems = node.getElementsByTagName("*");
        for(var i = 0, i < elems.length; i++){
            if (elems[i].className.indexOf(className) != -1) {
                results[results.length] = elems[i];
            }
        }
        return results;
    }
}

例子：

var shopping = document.getElementById("purchases");
var sale = getELementsByClassName(shopping,"sale");//获取购物列表

获取和设置属性

一、getAttribute–获取属性节点

getAttribute方法不属于document对象，不能通过document对象调用，只能通过元素节点对象调用

  • 方法：object.getAttribute(attribute)

  • 例子：

    var paras = document.getElementsByTagName("p");
for(var i = 0;i < paras.length;i++) {
    var title_text = paras[i].getAttribute("title");
    if (title_text) {
    //if(something)与if(something != null)完全等价，但前者更简洁
        alert(title_text);
    }
}

二、setAttribute–设置属性节点的值

和getAttribute一样，只能用于元素节点

  • 方法：object.setAttribute(attribute,value)

  • 例子：

    var shopping = document.getElementById("purchases");
alert(shopping.getAttribute("title"));//弹出空白或"null"
shopping.setAttribute("title","a list of goods");
alert(shopping.getAttribute("title"));//弹出"a list of goods"

var paras =  document.getElementsByTagName("p");
for(var i = 0;i < paras.length;i++) {
    var title_text = paras[i].getAttribute("title");
    if (title_text) {
        paras.setAttribute("title","brand new title text");
        alert(paras[i].getAttribute("title"));
    }
}

总结：

  1. 当要修改的属性不存在时，先创建这个属性，然后设置它的值；当元素节点上本来就有这个属性时，设置的值会覆盖原有值
  2. setAttribute做出的修改不会反映在文档本身的源代码里（DOM的工作模式：先加载文档的静态内容，再动态刷新，动态刷新不会影响文档的静态内容，即对页面内容进行刷新却不需要在浏览器里刷新页面）

    案例

    一、childNodes属性–用于获取任何一个元素的所有子元素

    返回的数组包含所有类型的节点，而不仅是元素节点，其中甚至包括空格和换行符等
  • 语法：element.childNodes

  • 例子：

    function countBodyChildren() {
    var body_element = document.getElementsByTagName("body")[0];
    alert(body_element.childNodes.length);
}//显示body元素的子元素的总个数

    二、nodeType属性

    每一个节点都有nodeType属性

  • 语法：node.nodeType

  • 例子：

    var body_element = document.getElementsByTagName("body")[0];
alert(body_element.nodeType);`
  • nodeType属性总共有12种可取值，仅3中具有实用价值
    1. 元素节点的nodeType属性值是1
    2. 属性节点的nodeType属性值是2
    3. 文本节点的nodeType属性值是3

CSS选择器

1.标签选择器（以文档元素作为选择符）

  • 语法：

    E {
    CSS规则
}

  • 示例：

    td {
    font-size:14px;
    width:120px;
}
a {
    text-decoration:none;
}

    2.ID选择器（以文档元素的唯一标识符ID作为选择符）

  • 语法：

    #ID {
    CSS规则
}

  • 示例：

    #note {
    font-size:14px;
    width:120px;
}

    3.类选择器（以文档元素的class作为选择符）

  • 语法：

    E.className {
    CSS规则
}

  • 示例：

    div.note {
    font-size:14px;
}

    4.群组选择器（多个选择符应用同样的样式规则）

  • 语法：

    E1,E2,E3 {
    CSS规则
}

  • 示例：

    td,p,div.a {
    font-size:14px;
}

    5.后代选择器（元素E的任意后代元素F）

  • 语法：

    E F {
    CSS规则
}

  • 示例：

    #link a {
    color:red;
}

    6.通用选择器（以文档的所有元素作为选择符）

  • 语法：

    * {
    CSS规则
}

  • 示例：

    * {
    font-size:14px;
}

（5）子元素过滤选择器

选择器 描述 返回 示例
:nth-child(index/even/odd/equation) 选取每个父元素下的第index个子元素或者奇偶元素（index从1开始） 集合元素

  • :first/:last只返回单个元素，而:first-child/:last-child将为每一个父元素匹配第一个/最后一个子元素；
  • nth-child的三种用法:
    1)nth-child(x)，获取第x个子元素；
    2) :nth-child(even)和:nth-child(odd)，从1开始，获取第偶数个元素或第奇数个元素；
    3) :nth-child(xn+y)，x>=0，y>=0。例如x = 3, y = 0时就是3n，表示取第3n个元素（n>=0）。实际上xn+y是上面两种的通项式。（当x=0,y>=0时，等同于:nth- child(x)；当x=2,y=0时，等同于nth-child(even)；当x=2,y=1时，等同于:nth-child(odd)）。
