CSS选择器

1.标签选择器（以文档元素作为选择符）

  • 语法：

    E {
    CSS规则
}

  • 示例：

    td {
    font-size:14px;
    width:120px;
}
a {
    text-decoration:none;
}

    2.ID选择器（以文档元素的唯一标识符ID作为选择符）

  • 语法：

    #ID {
    CSS规则
}

  • 示例：

    #note {
    font-size:14px;
    width:120px;
}

    3.类选择器（以文档元素的class作为选择符）

  • 语法：

    E.className {
    CSS规则
}

  • 示例：

    div.note {
    font-size:14px;
}

    4.群组选择器（多个选择符应用同样的样式规则）

  • 语法：

    E1,E2,E3 {
    CSS规则
}

  • 示例：

    td,p,div.a {
    font-size:14px;
}

    5.后代选择器（元素E的任意后代元素F）

  • 语法：

    E F {
    CSS规则
}

  • 示例：

    #link a {
    color:red;
}

    6.通用选择器（以文档的所有元素作为选择符）

  • 语法：

    * {
    CSS规则
}

  • 示例：

    * {
    font-size:14px;
}

    参考：https://segmentfault.com/a/1190000003088878#articleHeader5

jQuery选择器

1.基本选择器

选择器 描述 返回 示例
#id 根据给定的id匹配一个元素 单个元素 $(“#test”)选取id为test的元素
.class 根据给定的类名匹配元素 集合元素 $(“.test”)选取所有class为test的元素
element 根据给定的元素名匹配元素 集合元素 $(“p”)选取所有的<p>元素
* 匹配所有元素 集合元素 $(“*“)选取所有元素
selector1,selector2,…,selectorN 将每一个选择器匹配到的元素合并后一起返回 集合元素 $(“div,span,p.myClass”)选取所有<div>，<span>和拥有class为myClass的<p>标签的一组元素

2.层次选择器（通过DOM元素之间的层次关系来获取特定元素）

后代元素、子元素、相邻元素、同辈元素等

选择器 描述 返回 示例
$(“ancestor descendant”) 选取ancestor元素里的所有descendant（后代）元素 集合元素 $(“div span”)选取<div>里的所有<span>元素
$(“parent>child”) 选取parent元素下的child（子）元素，与$(“ancestor descendant”)有区别，$(“ancestor descendant”)选择的是后代元素 集合元素 $(“div>span”)选取<div>元素下元素名是<span>的子元素
$(“prev+next”) 选取紧接在prev元素后的next元素 集合元素 $(“.one+div”)选取class为one的下一个<div>同辈元素
$(“prev~siblings”) 选取prev元素之后所有同辈的siblings 集合元素 $(“#two~div”)选取id为two的元素后面的所有<div>同辈元素
  • 可以使用next()方法来代替$(“prev+next”)选择器，$(“.one+div”);等价于$(“.one”).next(“div”);；
  • 可以使用nextAll()方法代替$(“prev~siblings”)选择器，$(“#two~div”);等价于$(“#two”).nextAll(“div”);；
  • siblings()方法，$(“#prev”).siblings(“div”);选取#prev所有同辈的<div>元素。

3.过滤选择器

（1）基本过滤选择器

选择器 描述 返回 示例
:first 选取第一个元素 单个元素 $(“p:first”)选取第一个<p>元素
:last 选取最后一个元素 单个元素 $(“p:last”)选取最后一个<p>元素
:even 选取索引是偶数的所有元素，索引从0开始 集合元素 $(“tr:even”)所有索引是偶数的<tr>元素
:odd 选取索引是奇数的所有元素，索引从0开始 集合元素 $(“tr:odd”)所有索引是奇数的<tr>元素
:eq(index) 选取索引等于index的元素，index从0开始 单个元素 $(“ul li:eq(3)”)选取列表中的第四个元素（index 从 0 开始）
:gt(index) 选取索引大于index的元素，index从0开始 集合元素 $(“ul li:gt(3)”)选取索引大于3的元素
:lt(index) 选取索引小于index的元素，index从0开始 集合元素 $(“ul li:lt(3)”)选取索引小于3的元素
:not(selector) 选取所有与给定选择器不匹配的元素 集合元素 $(“input:not(:empty)”)选取所有不为空的<input>元素
:header 选取所有的标题元素<hx> 集合元素 $(“:header”) 选取所有标题元素<h1> - <h6>
:animated 选取当前正在执行动画的所有元素 集合元素 $(“div:animated”)选取正在执行动画的所有<div>元素
:focus 选取当前获取焦点的元素 集合元素 $(‘:focus’)选取当前获取焦点的元素

（2）内容过滤选择器

选择器 描述 返回 示例
:contains(text) 选取含有文本内容为“text”的元素 集合元素 $(“:contains(‘hello’)”)选取包含文本“hello”的所有元素
:empty 选取不包含子元素或者文本的空元素 集合元素 $(“:empty”)选取无子（元素）节点（包括文本元素）的所有元素
:has(selector) 选取含有选择器所匹配的元素的元素 集合元素 $(“div:has(p)”)选取含有<p>元素的<div>元素
:parent 选取包含子元素或者文本的元素 集合元素 $(“div:parent”)选取拥有子（元素）节点（包括文本元素）的<div>元素

（3）可见性过滤选择器

选择器 描述 返回 示例
:hidden 选取所有不可见的元素，包括<input type=”hidden”/>,<div style=”display:none;”>和<div style=”visibility:hidden;”>等元素 集合元素 $(“p:hidden”)选取所有隐藏的<p>元素
:visible 选取所有可见的元素 集合元素 $(“table:visible”)选取所有可见的表格

（4）属性过滤选择器

<div title="test">title为test的div元素①</div>
<div title="test-attr">title为test-attr的div元素②</div>
<div title="testing">title为testing的div元素③</div>
<div title="test attr">title为test attr的div元素④</div>
<div title="attrtest">title为attrtest的div元素⑤</div>
选择器 描述 返回 示例
[attribute] 选取拥有此属性的元素 集合元素 $(“[href]”)选取所有带有href属性的元素
[attribute=value] 选取属性值为value的元素 集合元素 $(“a[href=’#’]”)选取所有 href 属性的值等于 “#” 的<a>元素
[attribute!=value] 选取属性值不等于value的元素 集合元素 $(“div[title!=test]”)选取属性title不等于 “test” 的<div>元素（注意：没有属性title的<div>元素也会被选取）
[attribute^=value] 选取属性值以value开始的元素 集合元素 $(“div[title^=test]”)选取属性title以”test”开始的<div>元素①②③④
[attribute$=value] 选取属性值以value结束的元素 集合元素 $(“div[title$=test]”)选取属性title以”test”结束的<div>元素①⑤
[attribute*=value] 选取属性值含有value的元素 集合元素 $(“div[title*=test]”)选取属性title含有”test”的<div>元素①②③④⑤
[attribute|=value] 选取属性值等于给定字符串或以该字符串为前缀（该字符串后跟一个连字符“-”）的元素 集合元素 $(“div[title|=’test’]”)选取属性title等于”test”或以“test”为前缀的<div>元素①②
[attribute~=value] 选取属性用空格分隔的值中包含给定字符串的元素 集合元素 $(“div[title~=test]”)选取属性title用空格分隔的值中包含”test”的<div>元素④
[attribute1][attribute2][attributeN] 选取匹配所有属性选择器的元素 集合元素 $(“div[id][title$=test]”)选取拥有属性id，并且属性title以”test”结束的<div>元素

（5）子元素过滤选择器

选择器 描述 返回 示例
:nth-child(index/even/odd/equation) 选取每个父元素下的第index个子元素或者奇偶元素（index从1开始） 集合元素
:first-child 选取每个父元素的第1个子元素 集合元素
:last-child 选取每个父元素的最后一个子元素 集合元素
:only-child 如果某元素是它的父元素中唯一的子元素，则匹配，否则不匹配 集合元素
  • :eq(index)只匹配一个元素，而:nth-child将为每一个父元素匹配子元素，且前者index从0开始，后者从1开始；
  • :first/:last只返回单个元素，而:first-child/:last-child将为每一个父元素匹配第一个/最后一个子元素；
  • :nth-child(3n)能选取每个父元素下索引值是3的倍数的元素。

（6）表单对象属性过滤选择器

选择器 描述 返回 示例
:enabled 选取所有可用元素 集合元素 $(“input:enabled”)选取所有激活的 input 元素
:disabled 选取所有不可用元素 集合元素 $(“#form:disabled”)选取id为“form”的所有禁用的元素
:selected 选取所有被选中的选项元素（下拉列表） 集合元素 $(“select option:selected”)选取所有被选中的选项元素
:checked 选取所有被选中的元素（单选框，复选框） 集合元素 $(“input:checked”)选取所有被选中的<input>元素

4.表单选择器

选择器 描述 返回 示例
:input 选取所有的<input>、<textarea>、<select>和<button>元素 集合元素 $(“:input”)选取所有<input>、<textarea>、<select>和<button>元素
:text 选取所有的单行文本框 集合元素 $(“:text”)选取所有 type=”text” 的 <input> 元素
:password 选取所有的密码框 集合元素 $(“:password”)选取所有 type=”password” 的 <input> 元素
:radio 选取所有的单选框 集合元素 $(“:radio”)选取所有 type=”radio” 的 <input> 元素
:checkbox 选取所有的复选框 集合元素 $(“:checkbox”)选取所有 type=”checkbox” 的 <input> 元素
:submit 选取所有的提交按钮 集合元素 $(“:submit”)选取所有 type=”submit” 的 <input> 元素
:reset 选取所有的重置按钮 集合元素 $(“:reset”)选取所有 type=”reset” 的 <input> 元素
:button 选取所有的按钮 集合元素 $(“:button”)选取所有 type=”button” 的 <input> 元素
:image 选取所有的图像按钮 集合元素 $(“:image”)选取所有 type=”image” 的 <input> 元素
:file 选取所有的上传域 集合元素 $(“:file”)选取所有 type=”file” 的 <input> 元素
